======================================
Model Evaluation Toolkit (metk)
======================================

Compatibility
-------------

 * Works with Python 2.7 3.6


Dependencies
------------

 * `argparse <https://pypi.python.org/pypi/argparse>`_

Installation
------------

.. code:: bash

  pip install modelevaltoolkit

Usage
-----

Run

.. code:: bash
  
  metk.py --help
